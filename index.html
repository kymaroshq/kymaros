Kymaros Charts

Official Helm chart repository for Kymaros — continuous backup restore validation for Kubernetes.

Quick Start
# Add the repository
helm repo add kymaros https://charts.kymaros.io

# Install Kymaros
helm install kymaros kymaros/kymaros \
  --namespace kymaros-system --create-namespace

Restore Validation

Automatically restores backups into isolated sandboxes and validates they actually work.

Confidence Scoring

6-level scoring system (0-100) across pod health, HTTP, TCP, exec, and resource checks.

RTO Measurement

Measures real Recovery Time Objectives so you know exactly how long a restore takes.

Prometheus & Grafana

Built-in metrics exporter and Grafana dashboards for full observability.

Notifications

Slack and webhook alerts when restore tests fail or scores drop below thresholds.

SOC 2 Compliance

Generate compliance reports proving your backups are tested on schedule.
